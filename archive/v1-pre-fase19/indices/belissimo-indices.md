## SEÇÃO 8 — DESTAQUES PARA CALIBRAÇÃO

### O que torna este projeto especial:
1. **Alvenaria Estrutural** — projeto em sistema construtivo diferente da maioria na base (concreto armado). Permite calibrar custos de supraestrutura + alvenaria em sistema misto
2. **Escala massiva (180 un.)** — maior projeto da base em número de unidades, CUB ratio muito baixo (0,669 sem ger.) reflete ganho de escala
3. **Itajaí/SC** — mesmo mercado da Cartesian, custos alinhados com realidade local
4. **Comparativos incluídos** — a planilha já traz comparação vs paramétrico e obras similares, validando os índices
5. **Sistemas Especiais expressivos** — 4 elevadores, pressurização, exaustão — R$ 111,63/m² (vs mediana ~R$ 40/m²), puxando o macrogrupo pra cima
6. **Acabamento médio** — padrão porcelanato 60x60, laminado, alumínio L2.5 — referência boa para padrão médio residencial
7. **Custo total baixo** — R$ 1.983/m² (CUB ratio 0,756) é eficiente para o porte, demonstra ganho de escala na construção

### Pontos de atenção na calibração:
- **Não usar como referência direta para projetos pequenos** — o ganho de escala distorce os índices
- **Supraestrutura + Alvenaria devem ser lidos juntos** — separados não fazem sentido para comparação com projetos convencionais
- **Imprevistos baixos (1,51%)** — projetos maiores tipicamente têm contingência menor em %
