## ⚙️ Metodologia

**Fórmula:**
```
Valor Final (R$/m²) = Base R$/m² × Fator CUB × Fator Briefing Composto
```

**Onde:**
- **Base R$/m²:** Mediana calibrada do macrogrupo (mar/2026, base de 75 projetos)
- **Fator CUB:** CUB Atual / CUB Base = 3.150 / 3.150 = 1,00
- **Fator Briefing Composto:** Produto de todos os fatores das 25 perguntas do briefing

**Limitações:**
- Estimativa preliminar, não substitui orçamento executivo
- Não inclui: terreno, projetos, aprovações, marketing, impostos
- Faixas podem variar conforme características específicas do projeto
