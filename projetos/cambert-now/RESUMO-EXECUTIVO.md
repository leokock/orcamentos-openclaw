## ⚠️ CARACTERÍSTICAS QUE IMPACTAM CUSTO

### Fatores de Elevação (+)
- ✓ **3 subsolos (garagens)** → +60% Infraestrutura + Mov. Terra
- ✓ **Contenção cortina de estacas** → +40% Infraestrutura
- ✓ **Laje mista (protendido + armado)** → +3% Supraestrutura
- ✓ **Padrão Alto** → +20-45% em acabamentos
- ✓ **Fachada mista (tijolinho + ripado)** → +30% Fachada
- ✓ **Lazer completo (12 itens)** → +40% Complementares
- ✓ **Centro Itajaí (mobilização, logística)** → +5-10% indiretos

### Fatores de Redução (-)
- ✓ **Repetição de 17 pavimentos tipo** → -5-10% produtividade
- ✓ **Laminado em vez de porcelanato** → -15% piso privativo
- ✓ **Automação básica** → -30% Sist. Especiais
- ✓ **Sem fotovoltaicas** → -10% Sist. Especiais
