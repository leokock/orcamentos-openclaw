## Nota sobre CUB Regional

Para cidades fora de SC, o modelo ajusta automaticamente via Fator CUB (CUB Atual / CUB Base). Mas o CUB de referência muda por estado:
- SC: ~R$ 3.000 (base da calibração)
- SP: ~R$ 2.300
- RJ: ~R$ 2.300
- BA: ~R$ 1.900

Projetos fora de SC passam por normalização CUB→SC para entrar na base de calibração. Os índices percentuais (distribuição entre macrogrupos) são relativamente estáveis entre regiões.
